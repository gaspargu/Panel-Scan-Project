# Panel-Scan-Proyect
Software solution for monitoring field vehicles through the the scanning of dashboard data (odometer, fuel range, and other key indicators).
The solution consists of a mobile application designed for field operations, capable of storing data locally on the device when internet connectivity is unavailable. A centralized backend system processes and manages the data submitted by transport operators. Finally, an analytics dashboard is generated to provide alerts and monitoring capabilities, including notifications for vehicles with expired certifications or documentation.
